Problem Specification : Typecasting in java programming 

Implementation : 

class Exercise6{
	public static void main(String[] args)
	{
	char c1='g';
	System.out.println("The character is:"+c1);
	
	int int1=(int)c1;
	System.out.println("The character is:"+int1);
	
	double dl=10.09;
	System.out.println("The value is:"+dl);
	
	int int2=(int)dl;
	System.out.println("The value is:"+int2);
	
	double d2=int1;
	System.out.println("The value is :"+d2);
	}
}

Output : The character is g
         The character is 103
         The value is 10.09
         The value is 10
         The value is 103.0
